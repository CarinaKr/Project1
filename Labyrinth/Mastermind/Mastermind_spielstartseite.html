Mastermind

So funktioniert's:

Ziel es Spieles ist es den geheimen Farbcode zu knacken.
Der Code besteht aus vier der acht auszuwählenden Farben.
Beginne bei der untersten Reihe und setze vier der Farben in die vier Kästchen. Danach klicke auf "Check Row".
Nun erscheinen rechts neben dem Spielfeld schwarze und weiße Kreise.
Ein weißer Kreis bedeutet du hast eine richige Farbe auf dem richtigen Platz.
Ein schwarzer Kreis bedeutet du hast eine richtige Farbe auf dem falschen Platz.
Du hast zehn Versuche um den richtigen Code herauszufinden.

Viel Erfolg!
